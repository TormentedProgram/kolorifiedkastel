# kolorifiedkastel
 KDE Plasma 6 KRunner pastel plugin
 tweaked with claude for personal workflow use only

![Screenshot](./img1.png)
![Screenshot](./img2.png)

# Build & Install
```
git clone https://github.com/TormentedProgram/kolorifiedkastel.git && cd kastel
mkdir build && cd build
cmake .. && make
sudo make install && kquitapp6 krunner
```
